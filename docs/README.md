# 组件库


## 运行步骤

* 安装依赖 `$ yarn`

* 编译运行
  - 小程序 `$ yarn dev`
  - 网页 `$ yarn dev:h5`

* 生产构建
  - 小程序 `$ yarn build`
  - 网页 `$ yarn build:h5`

* 查看调试
  - 小程序 使用微信开发者工具打开 `dist/dev/mp-weixin`
  - 网页 通过浏览器访问 `localhost:9998`
